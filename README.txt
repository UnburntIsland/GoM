多頁式官網檔案說明

頁面：
- index.html 首頁
- about.html 品牌理念
- gameplay.html 玩法設計
- progress.html 開發進度
- team.html 團隊介紹
- contact.html 聯絡合作

共用資源：
- assets/css/main.css 主樣式
- assets/css/components.css 元件樣式
- assets/css/pages.css 頁面補充樣式
- assets/js/main.js 共用導覽/手機選單/捲動效果
- assets/js/gameplay.js 玩法頁 tabs
- assets/js/progress.js 進度頁篩選
- assets/js/contact.js 聯絡表單驗證
