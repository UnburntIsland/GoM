# 森循島｜永續科普教育遊戲官網

此資料夾為可部署的靜態網站。首頁為 `index.html`，可部署於 GitHub Pages、Netlify、Vercel、Cloudflare Pages 或一般靜態主機。

## 頁面

- `index.html` 首頁
- `about.html` 計畫理念
- `gameplay.html` 玩法科普
- `impact.html` 永續影響
- `progress.html` 執行規劃
- `team.html` 團隊介紹
- `contact.html` 合作聯絡
- `404.html` 錯誤頁

## 資產

- `assets/css/main.css`
- `assets/js/main.js`
- `assets/js/gameplay.js`
- `assets/js/progress.js`
- `assets/js/contact.js`
- `assets/images/`
